Standard closeout — apply at the end of every session
  1. For each completed item, on its <article id="…">: change data-status TODO→DONE, update visible pill text to DONE, set data-updated to today, add a one-line note.

  3. Update the project's session handoff file with the outcome.
  4. Log writes per the project's auto-write discipline.
  5. Hand off to the next session.
